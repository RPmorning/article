目录：

  1. 获取频道列表信息(首页)
  2. 搜索文章(带分页)
  3. 获取频道列表信息(列表页)(带分页)
  4. 获取部门签收状态
  5. 获取文章详情
  6. 增加文章浏览次数
  7. 部门签收文章
  8. 获取友情链接

4.获取部门签收状态

6.增加文章浏览次数

请求url:

/look

请求方式:

GET

参数类型:

参数 是否必选 类型 默认值 说明
articleId yes string / 当前被浏览文章所对应的Id

返回示例：

{
  status : 200,
  info : '增加浏览次数成功'
}

7.部门签收文章

请求url:

/sign

请求方式:

POST

参数类型:

参数 是否必选 类型 默认值 说明
articleId yes string / 当前被签收文章所对应的Id
username yes string / 部门用户名
password yes string / 部门登录密码

返回示例：

{
  status : 200,
  info : '签收文章成功'
}
